• Programacion
    1. Fundamentos: Tipos de datos, variables y operadores.
    2. Estructuras de control: Bucles (for, while) y condicionales (if, switch).
    3. Programación Orientada a Objetos (POO): Clases, objetos, herencia y polimorfismo.
    4. Estructuras de datos: Arrays, listas y colecciones.
  • Bases de Datos
    1. Modelo Entidad-Relación: Diseño lógico de los datos.
    2. SQL: Consultas, inserciones y actualizaciones de datos.
    3. Normalización: Reglas para evitar la duplicidad de información.
    4. Bases de Datos NoSQL: Introducción a MongoDB o similares.
  • Lenguaje de Marcas
    1. HTML5: Estructura de las páginas web (párrafos, listas, tablas).
    2. CSS3: Diseño, colores, fuentes y layout (Flexbox, Grid).
    3. XML y JSON: Formatos para el intercambio de datos.
    4. Sindicación de contenidos: RSS y Atom.
  • Computacion en la nube
    1. Modelos de servicio: SaaS, PaaS e IaaS.
    2. Proveedores principales: AWS, Azure y Google Cloud.
    3. Virtualización y Contenedores: Docker y Kubernetes.
    4. Almacenamiento y Seguridad: Gestión de datos en la red.
  • Sistemas Informaticos
    1. Sistemas Operativos: Instalación y configuración de Windows y Linux.
    2. Gestión de memoria y procesos: Cómo funciona el "cerebro" del PC.
    3. Redes: Protocolos TCP/IP, subredes y direccionamiento.
    4. Comandos: Uso de la terminal (Bash o PowerShell).
  • Itinerario Para la Empleabilidad
    1. Derecho Laboral: Contratos, nóminas y seguridad social.
    2. Prevención de Riesgos: Seguridad en el entorno de trabajo.
    3. Búsqueda de empleo: CV, entrevistas y marca personal.
    4. Emprendimiento: Creación de pequeñas empresas.
  • Entornos de Desarrollo
    1. IDEs: Configuración de Visual Studio Code, IntelliJ o Eclipse.
    2. Control de versiones: Uso profesional de Git y GitHub.
    3. Pruebas (Testing): Pruebas unitarias para asegurar que el código funciona.
    4. Diagramas UML: Planificación visual de las aplicaciones.
